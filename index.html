HU : telepítés / ENG: Install

Csomagold ki egy program könyvtárba. Ennek a könnyvtárnak az elérési utját állítsd be GROOVY_HOME környezeti változónak. A PATH környezeti változót bővítsd ki a %GROOVY_HOME%/bin bejegyzéssel.

Fordítás

groovyc Hello.groovy -- vagy tömeges forditás maszk alapján groovyc *.groovy *.java

Futtatás

java -cp %GROOVY_HOME%/embeddable/groovy-all-2.0.0.jar;. kl_class "Kecskemeti Lajos" ---------------------------------------------------------------------------
